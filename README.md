# 诗径 · 走到诗里

一款"地理位置触发古诗词"的体验原型：在西湖沿线的真实古道行走，走到某处点位，便弹出当年文人在**同一处**写下的诗句——还原"站在同一块石头上"的时空重叠感。

## 理念

- **位置触发，不是搜索**：走到了，它自己浮现，而非主动查询。
- **内容考据是魂**：只收"确切(A)"级——诗文有文献可考、确指此地此景此事。查不到确指文献的点位，宁可留白不触发，也不以意境相关之作填充。
- **诚实标注**：每条诗文附文献出处，便于复核。坐标为演示用近似值，正式上线前需实地打点校准。

## 已建诗径

1. **古龙井 · 风篁岭诗径**（4 点位）— 秦观《龙井题名记》、苏轼《次辩才韵》、米芾书《方圆庵记》
2. **孤山 · 白堤诗径**（6 点位，年代跨 1417 年）— 白居易《钱塘湖春行》、林逋《山园小梅》、苏小小歌、李贺《苏小小墓》

## 技术

- 单文件 `index.html` + `data/*.json` 内容库，无后端
- 地图：Leaflet + CartoDB Voyager 无字底图（WGS-84），叠宣纸滤镜
- 定位：`Geolocation API` + haversine 地理围栏；坐标统一存 WGS-84
- 诗径连线：本地 Catmull-Rom 样条平滑

## 本地运行

```bash
python3 -m http.server 8770
# 浏览器打开 http://localhost:8770
```
